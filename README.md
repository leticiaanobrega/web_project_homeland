# Tripleten web_project_homeland

# Nome do projeto:

De Pátria para Pátria

# Descrição:

Este projeto apresenta uma página web responsiva sobre diferentes cidades ao redor do mundo, explorando as histórias e culturas de diversos lugares através de uma galeria interativa. O site permite aos usuários conhecer diferentes destinos e suas características únicas através de um layout moderno e adaptativo.

# Tecnologias e técnicas utilizadas:

- HTML5 semântico
- CSS3 com metodologia BEM
- Flexbox para layouts flexíveis
- CSS Grid para estruturas complexas
- Web design responsivo com media queries
- Breakpoints para desktop (1280px), tablet (768px) e mobile (320px)
- GitHub Pages para hospedagem
